## 八、三个新系统的设计决策（2026-04-01更新）

### 8.1 法宝系统（替代武器系统）

**设计理念**：修仙世界的战斗依赖法宝（法器），而非原始武器

#### 法宝分类

| 法宝类型 | 代表法宝 | 效果 | 获取方式 |
|---------|---------|------|---------|
| **攻击型** | 骨矛 → 飞剑 → 玄冰剑 → 天雷剑 | 增加攻击伤害 | 炼制/遗迹 |
| **防御型** | 护体玉符 → 金光盾 → 玄铁甲 → 道基护符 | 增加护盾/减伤 | 炼制/遗迹 |
| **辅助型** | 灵兽袋 → 缩地符 → 天眼符 → 遁空符 | 特定场景增益 | 事件/遗迹 |

#### 法宝进阶树

```
骨矛（凡品）
├── 铁剑（良品）→ 飞剑（下品灵器）
│                     └── 玄冰剑（中品灵器）
│                              └── 天雷剑（上品灵器）
└── 兽皮甲（良品）→ 金光盾（下品灵器）
                    └── 玄铁甲（中品灵器）
                             └── 道基护符（上品灵器）
```

#### 法宝属性

```javascript
var ArtifactSystem = {
    // 法宝属性
    artifact: {
        type: 'weapon',      // weapon | armor | tool
        grade: 'mortal',    // mortal | spirit | divine
        damage: 10,          // 基础伤害
        durability: 100,     // 耐久度
        special: null        // 特殊效果ID
    }
};
```

#### Buff类型定义

```javascript
var BuffSystem = {
    // Buff配置
    buffs: {
        // 修炼加速（丹药）
        'cultivation_speed': {
            name: '小还丹',
            icon: '💊',
            type: 'potion',        // potion | buff | debuff
            duration: 300,         // 秒
            effect: 'cultivation_rate_2x',
            desc: '修炼速度翻倍',
            color: '#4CAF50'
        },

        // 突破加成（丹药）
        'breakthrough_boost': {
            name: '破境丹',
            icon: '✨',
            type: 'consumable',    // 一次性使用
            duration: 0,
            effect: 'breakthrough_rate_15',
            desc: '突破成功率+15%',
            color: '#FFC107'
        },

        // 渡劫辅助（丹药）
        'thunder_shield': {
            name: '天雷丹',
            icon: '⚡',
            type: 'stackable',     // 可叠加
            duration: 0,
            effect: 'block_thunder',
            stackable: true,
            desc: '抵挡一道天雷',
            color: '#9C27B0'
        },

        // 护体增益（被动/临时）
        'body_shield': {
            name: '护体符',
            icon: '🛡',
            type: 'buff',
            duration: 600,
            effect: 'shield_regen',
            desc: '每秒恢复护盾',
            color: '#2196F3'
        },

        // 减益状态
        'injury': {
            name: '经脉受损',
            icon: '💔',
            type: 'debuff',
            duration: 180,
            effect: 'cultivation_rate_0.5',
            desc: '修炼速度减半',
            color: '#F44336'
        }
    },

    // Buff管理
    activeBuffs: {},

    // 添加Buff
    addBuff: function(id, stacks) {
        var buff = this.buffs[id];
        if (!buff) return;

        if (buff.stackable) {
            this.activeBuffs[id] = (this.activeBuffs[id] || 0) + (stacks || 1);
        } else {
            this.activeBuffs[id] = 1;
            if (buff.duration > 0) {
                setTimeout(() => this.removeBuff(id), buff.duration * 1000);
            }
        }
        this.renderBuffBar();
    },

    // 移除Buff
    removeBuff: function(id) {
        delete this.activeBuffs[id];
        this.renderBuffBar();
    },

    // 渲染Buff栏
    renderBuffBar: function() {
        var html = '<div id="buff-bar">';
        for (var id in this.activeBuffs) {
            var buff = this.buffs[id];
            var stacks = this.activeBuffs[id];

            html += '<div class="buff-item" title="' + buff.desc + '">';
            html += '<span class="buff-icon">' + buff.icon + '</span>';

            if (buff.type === 'stackable') {
                html += '<span class="buff-stacks">×' + stacks + '</span>';
            } else if (buff.duration > 0) {
                html += '<span class="buff-timer">' + this.formatTime(buff.duration) + '</span>';
            }

            html += '</div>';
        }
        html += '</div>';
        // 插入到header区域
    }
};
```
