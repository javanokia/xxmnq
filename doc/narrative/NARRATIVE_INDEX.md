## 叙事架构快速导航

### 主线四幕

```
第一幕：洞中独坐（Room模块）
  ↓  builder.level >= 4
第二幕：荒野行路（Outside / World模块）
  ↓  境界凝元+
第三幕：凝元归虚（深层修炼阶段）
  ↓  builder.level >= 6
第四幕：天劫渡劫（Space模块）
  ↓  
  成功 → 渡劫结局
  失败 → 未竟结局
  轮回 → Prestige → 前世余韵 → 新一轮开始
```

### 主题问题
> 在灵气散尽、天道已绝的世界，一介凡人，修炼是意义还是执念？

## 世界圣经·禁止修改的第一层设定

1. 老散修来自北方某废弃宗门（名字永不揭晓）
2. 游戏主角无名无姓（永远用"他"指代）
3. 末法的原因未知（永不揭晓）
4. 游戏发生在末法第五十年

## 叙事风格检查清单

编写或审查任何文案时，对照以下清单：

- [ ] 每行文字都是必要的吗？（删到不能再删）
- [ ] 是否用了禁用词：豁然开朗/心中一动/莫名感到/不知为何
- [ ] 角色对话是否符合对应的声优档案？
- [ ] NPC是否对玩家做了道德评判？（不该有）
- [ ] 是否有"伪装成对话的说明"？（必须删除）
- [ ] 玩家的选择是否有可感知的后果？
