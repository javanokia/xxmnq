# 《一介凡尘》叙事设计总纲

> **版本**：v1.0  
> **创建**：2026-04-01  
> **叙事设计原则**：极简文本·凡俗克制视角·行动即叙事

---

## 一、中心主题问题

> **"在灵气散尽、天道已绝的世界，一介凡人，修炼是意义还是执念？"**

这个问题不会被明确回答。玩家的每一次选择——收留流民还是保存物资、传授心法还是闭门自守、渡劫还是放弃——都是在用行动作答。

**副主题**：
- 末法时代，"道"还在吗？
- 一个人能改变多少？应该改变多少？
- 轮回之后，你还是你吗？

---

## 二、叙事风格规范

### 散文原则
- **克制**：描述事实，不评判情绪。让玩家自己感受。
- **密度**：每行文字必须赚到它的位置。删到不能再删。
- **视角**：第三人称，略带距离感。"他看着"而非"你感到"。
- **时态**：现在时。"火在燃烧。"不是"火曾经燃烧着。"
- **禁用词**：豁然开朗、心中一动、莫名感到、不知为何（直接写后果）

### 参考腔调
```
✓ 好的示例：
"火小了一些。他没有添柴。"
"门口站着一个人，背对着光。"
"灵晶用完了。还剩三天的路。"

✗ 不好的示例：
"你豁然开朗，感受到了修炼的真谛！"
"神秘的力量流遍全身，让你充满了希望。"
"归隐老修慈祥地笑了笑，说道……"
```

## 三、主线叙事框架（四幕结构）

### 第一幕：洞中独坐（Room模块）
**情感基调**：陌生 → 熟悉 → 轻微依赖

**叙事节拍**：
1. **起点**：山洞，黑暗，炉火将熄。一个老人。没有名字，没有背景。
2. **转折**：第一个陌生人出现在洞口——不是威胁，只是寒冷中的另一个人。
3. **节奏**：散修陆续到来。营地形成。主角开始有了角色：守火者。
4. **小危机**：第一次资源危机，或第一次内讧，或第一次妖兽袭击。
5. **幕末问题**：这火，是要烧给别人看的吗？

**幕末叙事触发条件**：`game.builder.level >= 4`（解锁Outside）

**幕末文案**：
```
洞已经太小了。
他在石壁上划了一道印记，算是告别。
老散修没有多说。只是把手炉递过来——"路上冷。"
```

---

### 第二幕：荒野行路（Outside/World模块）
**情感基调**：开阔 → 残酷 → 理性接受

**叙事节拍**：
1. **出发**：第一次走出去，草木皆兵。
2. **发现**：废弃坊市、倒塌的宗门山门、无名的修士残骸。
3. **真相**：末法不是传言。修炼者大多已死或散去。
4. **人性**：劫修、拾荒者、孤村凡民——各有各的活法。
5. **幕末问题**：继续走，还是回去守着那堆火？

**关键环境叙事场景**（详见第六节）：
- 倒塌的宗门山门
- 废弃炼丹房
- 布满尸骨的道路
- 孤村老者

**幕末文案**：
```
大陆比预想的更大，也更空旷。
活着的人越来越少，烽火台上的灵火早已熄灭。
他找到了一本残卷，上面写着渡劫之法——
字迹已被血迹模糊。
```

---

### 第三幕：凝元归虚（深层修炼阶段）
**情感基调**：沉静 → 孤独 → 某种清醒

**叙事节拍**：
1. **境界压力**：境界越高，世界越沉默。高修者独处的代价。
2. **决定时刻**：营地已不需要他了。人们已经可以自己决定事情。
3. **最后问道**：老散修再次出现（若玩家曾收留他的传承线）。
4. **幕末**：天劫将至。不是选择，是必然。

**幕末文案**：
```
某日清晨，天色异常。
积云如铁，低压在修炼之地上方。
有人说，百年未见这种天象。
他知道那是什么。
```

---

### 第四幕：天劫渡劫（Space模块）
**情感基调**：绝境 → 专注 → 无论结果

**叙事节拍**：
1. **最后凝视**：出发前，营地最后一瞥。
2. **天界**：一片虚空，只有他和天雷。
3. **结局A·渡劫成功**：不是仙，是另一种孤独。
4. **结局B·渡劫失败**：死亡也是一种答案。
5. **结局C·轮回**：另一个洞穴，另一堆火，另一个自己？

**幕末文案（成功）**：
```
天雷第九击落下时，他已经不害怕了。
不是因为更强大。
是因为——这就够了。
```

**幕末文案（失败）**：
```
最后一道雷击中了天舟。
火散了。
他想起洞里还有半桶灵晶没用完。
```

**幕末文案（轮回触发）**：
```
意识消散之前，他看见了——
另一处山洞。
另一堆快要熄灭的火。
```

## 四、情感弧线图

```
情绪强度
  ↑
  │                          ●渡劫巅峰
  │                    ●凝元
  │          ●荒野残酷
  │   ●流民  
  │●初火  
  └─────────────────────────────→ 时间
  洞中独坐  荒野行路  归虚   天劫
```

**核心情绪旅程**：
- 开始时：孤独但不绝望
- 中途：温暖（营地形成）→ 惊醒（世界的残酷）
- 末途：宁静的孤独（不是悲伤的孤独）
- 结局：无论成败，都是完整的

## 五、叙事-游戏玩法集成矩阵

| 故事节拍 | 游戏玩法后果 | 玩家感受 |
|---------|------------|---------|
| 第一个流民到来 | 解锁茅庐建造 | 责任感 |
| 老散修传法（事件） | 修炼速度微升 | 被承认 |
| 资源危机（事件） | 必须做取舍选择 | 压力·紧迫 |
| 营地内讧（事件） | 人口减少 | 代价感 |
| 荒野发现废弃坊市 | 解锁特定配方 | 失落·惊喜 |
| 劫修袭击（外部） | 物资损失 | 世界是危险的 |
| 老散修最后一次出现 | 传承支线完结 | 告别·接纳 |
| 渡劫飞行前 | 锁定所有建造动作 | 这一刻无法回头 |
| 渡劫成功 | 进入轮回界面 | 开阔·孤独 |

## 六、世界圣经快速参考

### 时间线（末法元年起算）

| 时间 | 事件 |
|------|------|
| 末法元年 | 天道断裂，灵气开始稀薄。宗门率先感知异变。 |
| 末法第三年 | 各大宗门开始争夺残余灵脉。第一次修仙战争爆发。 |
| 末法第十年 | 大部分宗门崩解。散修时代开始。 |
| 末法第三十年 | 凡俗世界与修仙世界界线模糊。大量凡民死于余波。 |
| 末法第五十年（游戏现在） | 天地灵气仅余十之一二。渡劫者寥寥无几。 |

### 世界规则（不可违反）
1. **末法世界不允许飞行**：飞行需消耗大量灵力，普通散修无法持续。天舟是特例（消耗极大）。
2. **炼丹丹方已大量失传**：玩家找到的任何丹方都是残卷，不完整是常态。
3. **渡劫是末法时代最后的奇迹**：五十年来，成功渡劫者不超过十人。
4. **灵晶不可再生**：游戏中的"采集"是挖掘古战场遗留的结晶，总量有限。

### 禁止追溯修改的第一层设定
- 老散修来自北方某废弃宗门（名字不重要）
- 游戏主角无名无姓（永远用"他"指代）
- 末法的原因未知，也永远不会揭晓

## 七、传说层级结构

### 第一层：表面（所有玩家必然接触）
- 主角是一个在山洞里修炼的人
- 老散修教他点火（打坐）
- 山洞外面的世界一片萧瑟
- 最终要面对天劫

### 第二层：参与（探索者发现）
- 老散修曾是某宗门长老（通过"寒夜问道"后续事件透露）
- 营地里某些弟子有来历（随机事件中有线索）
- 废弃坊市里有战争遗留的物资，暗示当年规模
- 劫修中有曾经的修士（对话文案有暗示）

### 第三层：深层（传说猎人专属）
- 老散修是末法第一批渡劫失败者之一（前世余韵系统暗示）
- 轮回不是比喻——玩家Prestige时的"前世余韵"是字面意义的前世记忆
- 游戏世界循环本身是一种"末法天道的遗存机制"（meta层）
