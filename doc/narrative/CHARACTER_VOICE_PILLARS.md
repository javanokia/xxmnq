# 《一介凡尘》角色声优档案

> **用途**：NPC对话的权威参考。每个角色都有一致的声音。  
> **方法**：所有对话都必须通过"这个人会这样说吗？"测试。

---

## 角色一：归隐老修

### 身份定位
- **故事角色**：导师 + 镜像
- **核心创伤**：末法时代见证了整个修仙世界的崩溃。他或许也是渡劫失败者（前世余韵系统暗示）。
- **欲望**：传承。把知识传给还有热情的人。
- **需求**：原谅。为他曾见证的一切。

### 声优支柱
- **词汇**：古风、准确、少评判。"北方宗门的老腔调"。很少用感叹词。
- **句子节奏**：短句 + 长句交替。短句表示决断或观察，长句表示深思。
- **他避免的话题**：
  - 自己的过去（如果追问，只会说"那已经过去了"）
  - 对玩家的评价（从不说"你很不错"，只说"继续坐着"）
  - 道德判断（对玩家的选择不评价好坏）
- **语言习惯**：
  - 用"他"而非"你"来指代玩家（虽然ADR系统限制，但在事件中要体现）
  - 重复关键词强调意思（"火要烧，人要活。活着烧火，火才有意义。"）
  - 很少说"不"，常说"随他去"或"那是他的选择"
- **潜台词默认**：总是用暗示而非直说。"那女娃拿着灵晶，神情有些急。大概她有急用。"

### 他永远不会说的话

❌ **"你看起来很有天赋，一定能突破到开脉境！"**
- 原因：太直白，太鼓励。老修不这样说话。

❌ **"相信我，修炼之道就是沉着心性，体悟天地……"**
- 原因：太书生气，太哲学。老修讲的是活法，不是理论。

❌ **"我的徒儿啊，你一定不要放弃！"**
- 原因：太温情脉脉，不符合末法时代的沉静。

### 参考台词集（已通过声优审查）

```
"火灭了。添柴。"
— 极简，直指行动。没有附加情绪。

"那些灵晶是留来做什么的？是留来放着的吗？"
— 用问句推动思考，不直接命令。

"人来了三个。茅舍只够两间。有人要去睡洞外。"
— 客观叙述，暗示问题存在，让玩家自己解决。

"归虚之后，就没什么好说的了。自己去走，去看，去决定。"
— 长短混杂。承认生命有新的阶段，退出舞台。

"老了。记不清了。但灵晶什么的，吃得还是有点道理的。"
— 自嘲但不矫情。"有点道理"是保留意见的表达。

"你想渡劫。我不劝。那是活着最后的样子，也可能是死的样子。"
— 不评判。承认两种可能都存在。
```

## 角色二：求道者（女，20代）

### 身份定位
- **故事角色**：寄托者 + 希望的代价
- **核心创伤**：成长在末法时代，从未见过"修仙强盛"的时代。她问的"修炼之道"可能只是想活下去。
- **欲望**：学到足以自救的东西
- **需求**：被听见（而非被教导）

### 声优支柱
- **词汇**：年轻、不稳定、有求知欲但也有现实感。不用"大人""前辈"等敬称，会用"您"但很少。
- **句子节奏**：快。思维跳跃。时常自我打断。
- **她避免的话题**：
  - 自己的家人（如果涉及，会转移话题）
  - 对修炼的理想化想象
  - 承认害怕（会否认或转化为其他情绪）
- **语言习惯**：
  - 常用"那"和"然后"来连接想法
  - 会有感叹词，但多是"唉""哼"而非"哎呀"
  - 时常停顿，仿佛在思考是否该继续
- **潜台词默认**：她说的话不总是真心的，有时只是在测试。

### 她永远不会说的话

❌ **"感谢您的教诲！我一定会好好修炼，成为强大的修仙者！"**
- 原因：太幼稚，太理想主义。这个女孩长在末法时代。

❌ **"我听说您是某宗门的高人，是真的吗？"**
- 原因：她不会刻意恭维。如果知道，会自然地反应；不知道，就不问。

❌ **"我已经下定决心了，无论多苦都要修炼！"**
- 原因：太决绝。21世纪的年轻人不这样说话，即使在修仙世界也不会。

### 参考台词集

```
"那个……你是不是就在这儿坐着修炼？怎么坐啊？"
— 真诚的问，带点局促。不是装作不知道。

"有人说，修炼到一定程度就不用吃饭了。这是真的吗？我家里……"
— 停顿了。原想说家里没吃的，没说出来。含蓄。

"不对，我不是想……你别误会。我是想知道，如果——"
— 不完整的句子。她在临时改变表述。很真实。

"嗯。我记住了。然后呢？还有吗？"
— 渴求但也想让对方知道她在听。每次"然后"都是赌注。

"您说得……可能是对的。我回去试试。对了，两天后我再来。"
— 不承诺永远，只承诺两天。这是她能给的最多。
```

## 角色三：流浪修士

### 身份定位
- **故事角色**：过客 + 反镜
- **核心创伤**：曾是更高的修为，但某次渡劫/战斗中失去了什么（具体不说）。
- **欲望**：还能帮得上忙的地方
- **需求**：不被问起过去

### 声优支柱
- **词汇**：风尘气，有些慵懒。用词看似随意，实则精确。
- **句子节奏**：长句居多。似乎有故事，但不急着讲。
- **他避免的话题**：
  - 自己的修为现状
  - 为什么流浪
  - 特定的地名和宗门名字
- **语言习惯**：
  - 常用"听说""据说"来表达信息来源
  - 很少主动说"我"，更多用"一个人"来指自己
  - 笑声（但笑声中有苍凉）
- **潜台词默认**：他的故事永远没有结尾。对话中突然沉默，是正常的。

### 他永远不会说的话

❌ **"我曾经是某宗门的弟子，我会把我所有的经验都教给你！"**
- 原因：背景暴露。他不会这样坦白。

❌ **"我已经没什么好活的了，找个地方落脚就行。"**
- 原因：太自怜。再狼狈的修士也有尊严。

### 参考台词集

```
"听说东边有座山，山下的矿脉还不错。一个人去太危险。"
— 信息传递，同时暗示他曾去过。

"我在外头走了……（停顿）……很久。累了。你这儿还有位置吗？"
— 停顿很关键。他在决定说多少。

"年纪大了，反应慢了。但还能砍柴。"
— 自我评价，但带着某种自嘲的诚实。

"有人说末法之前，宗门高手一根手指就能挪山。现在——"
— 说到一半没说完。暗示他亲眼见过巨大的落差。
```

## 角色四：流民（泛指）

### 身份定位
- **故事角色**：依赖者 + 伦理困境来源
- **核心创伤**：没有太多外表化的创伤。他们就是"活着"这件事的困难示范。
- **欲望**：不死
- **需求**：被当作人看待（而不是劳动力）

### 声优支柱（这个群体没有一致的个人声音，但有共同特点）
- **词汇**：参差不齐。可能是凡民，也可能是破落修仙者。用词反映身份模糊。
- **句子节奏**：犹豫。常常不完整。
- **特点**：
  - 很少主动提要求
  - 会用"大人"之类的敬称，但很生硬
  - 常常转向询问而非陈述
- **潜台词默认**：他们在测试你的反应，看是否会赶他们走。

### 参考台词集

```
"能……能借个地儿待两晚吗？我们有点干妖髓可以换。"
— 不是乞讨，是交易提议。保留尊严。

"您这儿人多了。我们不想添麻烦。"
— 正在观察态度。为离开留伏笔。

"那个女娃问，为什么要坐着不动？我说……我不知道。"
— 叙述中带着某种无奈的诚实。
```

## 角色五：妖兽（拟人化呈现，仅限对话文案）

### 身份定位
- **故事角色**：环境威胁 + 共存者
- **特点**：虽然是"敌对"NPC，但也在末法时代挣扎活着

### 声优支柱
- **特点**：没有人类语言，只有行为描述和声音词。
- **呈现方式**：通过玩家的观察来"说话"

### 参考文案集

```
❌ 直接出现（不好）：
"一只妖兽冲进了你的营地，发出凶狠的吼声！"

✓ 用环境叙事（好）：
"夜里，有东西在翻掘储物间。
咆哮声很大，但听起来……很急。
大概是饿的。"

✓ 允许的互动文案：
"妖兽退走了，一摇一晃地消失在黑夜里。
身上少了块皮毛，看起来有点瘸。
也许它也在躲什么。"
```

## 角色六：天劫/天道（拟人化意志）

### 身份定位
- **故事角色**：最终试卷
- **特点**：不是邪恶的，只是一种自然的机制。无情，但公正。

### 呈现方式
- **不能说话**（它是天，天不需要说）
- **通过天象、天声、天雷来表达**
- **暗示逻辑而非道德**

### 参考文案集

```
"不是冲下来。只是天色变了。
如果说昨天的天是灰的，今天就是黑的。
而明天……你看不到明天的天。"

"天雷没有声音——或者说，声音太大了，反而听不见。
只看见光，还有烧焦的空气。"

"第九次闪电。这也许是最后一次。
也许还有第十次。
他已经分不出区别了。"
```

## 对话编写检查清单

在编写任何涉及以上角色的对话时，逐项检查：

- [ ] 这句话有没有"豁然开朗""莫名感到"这种描述词？（删掉）
- [ ] 这个角色在这个场景会这样说话吗？（对号入座声优支柱）
- [ ] 我有没有用"你"直接指代玩家？（改成行为或情境描述）
- [ ] 是否有任何"伟大"或"崇高"的言辞？（末法时代的对话应该接地气）
- [ ] NPC是否在给玩家作道德评判？（不该有。只描述后果。）
- [ ] 句子长度是否符合这个角色？（参考"短长交替"原则）
