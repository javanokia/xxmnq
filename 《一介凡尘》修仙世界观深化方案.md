### 2.2 境界显示改造

在游戏界面（`header.js`）中，增加境界显示：

```javascript
// header.js 改造：增加境界显示模块
renderCultivation: function() {
    var realm = Cultivation.getCurrentRealm();
    var level = $SM.get('game.builder.level', true);  // 境界内的小重天
    var realmIndex = Cultivation.getRealmIndex(realm.id);
    
    var html = '<div id="cultivation-status">';
    html += '<div class="realm-name">' + realm.name + '</div>';
    html += '<div class="realm-stage">' + realm.三重[level % 3] + '</div>';
    html += '<div class="realm-progress">';
    
    // 显示下一境界（灰色）
    if (realmIndex < Cultivation.realms.length - 1) {
        var nextRealm = Cultivation.realms[realmIndex + 1];
        html += '<span class="next-realm">' + nextRealm.name + '</span>';
    } else {
        html += '<span class="peak-realm">道之极</span>';
    }
    
    html += '</div></div>';
    return html;
}
```

## 三、修仙化资源体系

### 3.1 资源重定义

| 资源ID | 现有名称 | 修仙化名称 | 修仙化意义 |
|--------|---------|-----------|-----------|
| wood | 灵晶 | **灵晶** | 灵气结晶，一切修炼的基础 |
| meat | 妖肉 | **妖髓** | 妖兽精血，大补但有副作用 |
| fur | 兽皮 | **兽甲** | 防御材料，炼器用 |
| iron | 生铁矿 | **凡铁** | 最基础的炼器材料 |
| steel | 精铁 | **玄铁** | 较好的炼器材料 |
| coal | 阴火石 | **阴火煤** | 火属性炼器/炼丹材料 |
| sulphur | 焚晶砂 | **焚晶砂** | 火+毒属性材料 |
| scales | 妖鳞 | **妖鳞** | 妖兽材料，防御向 |
| teeth | 兽齿 | **兽牙** | 攻击向妖兽材料 |

## 七、叙事融入原则

### 7.1 核心原则

**修仙不是装饰，是机制。**

每一项修仙元素，都必须有对应的机制支撑：

```
概念层：结丹境
    ↓
叙事层："金丹在丹田中旋转，痛苦而缓慢。"
    ↓
机制层：玩家获得"金丹"状态 → 显示金丹图标 → 战斗中获得金丹保护加成
    ↓
结果层：境界突破后，玩家的游戏体验发生实质性变化
```

### 7.2 数值叙事化

将数值变化包装为叙事事件：

```javascript
// 示例：境界突破时的数值变化包装
onBreakthrough: function(newRealm, newLevel) {
    
    // 叙事优先：先显示叙事，再更新数值
    Events.trigger('cultivation_breakthrough', {
        realm: newRealm,
        level: newLevel,
        narrative: newRealm.breakthroughNarrative
    });
    
    // 数值更新（带延迟，让叙事先呈现）
    setTimeout(function() {
        $SM.set('game.builder.level', newLevel);
        $SM.set('game.health', 100);  // 突破后回满血
        $SM.add('stores.medicine', -newRealm.突破消耗.medicine);
    }, narrative_duration);
}
```

## 九、总结

修仙世界观的融入，不是"把火改成灵火、把木头改成灵石"这么简单。

**真正的修仙化需要三个层次：**

1. **叙事层** — 用修仙的语境描述一切：突破叫"顿悟"、受伤叫"经脉受损"、探索叫"寻幽探秘"
2. **机制层** — 每一种修仙元素都有对应的游戏机制：境界突破、丹药服用、法器炼制
3. **体验层** — 境界提升时要有仪式感，天劫来临时要有压迫感，渡劫成功时要有升华感
