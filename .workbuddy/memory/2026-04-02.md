# 工作日志 · 2026-04-02

## 修复 3 个运行时错误

**问题来源**：微信开发者工具真机预览报错

### 修复内容

1. **WXML 可选链语法错误** `exploration/index.wxml`
   - 错误：`{{regions[currentRegion]?.name || ''}}`
   - 修复：在 JS 的 `loadData()` 里预先计算 `currentRegionName`，WXML 直接用 `{{currentRegionName}}`
   - 原因：WXML 模板不支持 `?.` 可选链语法（JS 文件里可以用）

2. **音频 `play()` 不返回 Promise** `services/audio.js`
   - 错误：`Cannot read property 'then' of undefined`
   - 修复：`resumeBGM()` 和 `playSFX()` 直接调用 `play()`，不再链式调用 `.then()/.catch()`
   - 原因：`createInnerAudioContext()` 的 `play()` 方法不返回 Promise

3. **音频文件复制路径报错** `services/audio.js`
   - 错误：`copyFile:fail parameter error: parameter.destPath should be String instead of Undefined`
   - 修复：放弃复制到 USER_DATA_PATH 的方案，改为直接使用包内路径 `/assets/audio/xxx.mp3`
   - 原因：`wx.env.USER_DATA_PATH` 在构造函数执行时为 undefined

4. **app.json 无效字段** 
   - 删除了无效的 `"userInfo": {...}` 配置项
